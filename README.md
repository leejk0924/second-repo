# second-repo

## This is my second repository to practice how to start git with `$ git init`

## commands

```shell
$ mkdir second-repo
$ cd second-repo
$ git init
// after creating remote repo on github,
$ git remote add origin {repo_url}

// After doing some work,

$ git add {filename}
$ git commit

$ git branch -M main

$ git push -u origin main

```
